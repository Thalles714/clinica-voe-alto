# Clínica Voe Alto — Landing Page Institucional

Landing page premium desenvolvida para a **Clínica Voe Alto**, uma clínica multidisciplinar voltada ao atendimento de crianças, adolescentes, adultos e famílias.

O projeto foi criado com foco em **conversão, experiência do usuário, identidade visual, performance e presença digital profissional**, com o principal objetivo de gerar contatos qualificados pelo WhatsApp.

---

## Sobre o projeto

Este site foi desenvolvido como uma solução digital personalizada para apresentar os serviços da Clínica Voe Alto de forma clara, moderna e acolhedora.

A página foi planejada para conduzir o visitante por uma jornada simples:

1. Entender a proposta da clínica;
2. Conhecer os diferenciais;
3. Visualizar as especialidades;
4. Ver a estrutura e o ambiente;
5. Entender como funciona o primeiro contato;
6. Tirar dúvidas frequentes;
7. Iniciar uma conversa pelo WhatsApp.

---

## Objetivo da landing page

O principal objetivo deste projeto é transformar visitantes em contatos reais para a clínica.

A estratégia da página foi construída com foco em:

- clareza na comunicação;
- design moderno e acolhedor;
- navegação simples;
- boa experiência em celular e desktop;
- chamadas para ação bem posicionadas;
- integração com WhatsApp;
- SEO técnico básico;
- estrutura preparada para futuras melhorias.
