# Clínica Voe Alto

Landing page institucional desenvolvida para a **Clínica Voe Alto**, com foco em apresentar seus atendimentos multidisciplinares e facilitar o contato de pacientes e familiares pelo WhatsApp.

🌐 **Site publicado:** [clinicavoealto.com.br](https://clinicavoealto.com.br/)

## Tecnologias

- React
- Vite
- Tailwind CSS
- JavaScript
- GitHub Actions
- GitHub Pages

## Principais objetivos

- Experiência responsiva para desktop e mobile
- Conversão de visitantes pelo WhatsApp
- Apresentação clara das especialidades
- SEO técnico e SEO local
- Acessibilidade e performance
- Interface moderna, acolhedora e profissional

## Desenvolvimento

Projeto desenvolvido por **Thalles Leal**, com atuação em desenvolvimento de landing pages, sites institucionais, UX/UI, SEO técnico e soluções digitais para negócios locais.
