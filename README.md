# Clínica Voe Alto — Landing Page

Landing page premium desenvolvida para a Clínica Voe Alto, com foco em conversão via WhatsApp, SEO local e futura integração com ferramentas de leads.

## Stack

- React
- Vite
- Tailwind CSS
- GitHub Pages

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```
